# Apure' website update

This folder is a complete static website ready for GitHub Pages, Netlify, Cloudflare Pages, or a standard web host.

## Pages

- `/` - Homepage
- `/What-we-do/` - Detailed services
- `/Who-we-are/` - Founder-led company story
- `/Get-in-Touch/` - Hiring process and project inquiry

## Before launch

1. Replace the email-based inquiry action with the preferred CRM, form endpoint, or scheduling link if available.
2. Confirm the package prices and the public use of all project images.
3. Add verified client names, testimonials, and measurable outcomes when permission is available.
4. Submit `sitemap.xml` in Google Search Console and Bing Webmaster Tools.
5. Test the deployed URL with PageSpeed Insights and the Rich Results Test.

The site intentionally uses plain HTML, CSS, and JavaScript with no framework or third-party UI dependency.
